Api para Gerenciamento de Patrimônio

    Configurações

  1. Java 11
  2. Maven
  3. Projeto Criado no Site Spring initalzr
  4. Banco de Dados : MySQL

Detalhes para Consumo da Api

Api Usuário

Método GET usuario por Id

URl : http://localhost:8080/usuario/id

Resposta :

{
"id":"id",
"nome":"nome",
"email":"email"
}
